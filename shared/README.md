# Shared

This package contains the TypeScript contracts shared by the frontend and backend.

## Contents

- model capability types
- UI control specs
- telemetry event types
- agent plan and result types
- preflight gate inputs and outputs

## Purpose

Keep request and response shapes in one place so the frontend and backend do not drift apart.

## Files

- `index.ts` - shared interfaces and type definitions
- `package.json` - package metadata
